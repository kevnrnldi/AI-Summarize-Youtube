
Fase 8 Setup Frontend React :
[Done] Inisialisasi React + Vite: Buat project baru dengan perintah
    npm create vite@latest client -- --template react

[Done] Instalasi Package Frontend: Install axios (HTTP request),
    react-router-dom (navigasi halaman), dan tailwindcss (styling)

[Done] Konfigurasi Proxy: Di file vite.config.js, arahkan semua
    request /api ke backend Express di localhost:3000 agar tidak
    kena CORS error

[Done] Arsitektur Folder React:
    src/
    ├── pages/        ← halaman-halaman utama
    ├── components/   ← komponen kecil yang bisa dipakai ulang
    ├── utils/        ← fungsi helper (sessionId, axios instance)
    └── hooks/        ← custom hooks



Fase 9 Manajemen Sesi Pengguna (Tanpa Login)k
[Done] Buat SessionId Generator: Di src/utils/session.js, buat fungsi
    yang cek localStorage — kalau belum ada sessionId maka generate
    UUID baru pakai crypto.randomUUID() dan simpan

[Done] Buat Axios Instance: Di src/utils/api.js, buat axios instance
    yang otomatis menyertakan sessionId di setiap request header
    sehingga tidak perlu tulis manual di tiap komponen

[Done] Test Sesi: Buka browser, cek localStorage di DevTools —
    pastikan sessionId muncul otomatis saat halaman pertama dibuka



Fase 10 Halaman Utama 
[Done] Buat Halaman Home: Di src/pages/Home.jsx, buat tampilan dengan:
    - Input field untuk URL YouTube
    - Tombol "Analisis Video"
    - Loading state saat menunggu response backend

[Done] Integrasi API Analyze: Hubungkan tombol ke POST /api/analyze,
    kirim URL yang diinput user, tangkap response summary dari backend

[Done] Navigasi ke Halaman Video: Setelah dapat response berhasil,
    arahkan user ke halaman video dengan membawa videoId
    menggunakan react-router-dom useNavigate

[Done] Handle Error: Tampilkan pesan error yang ramah jika URL tidak
    valid atau video gagal dianalisis

Fase 13 Sistem Login
[ ] Backend — Buat Model User: Tambah UserSchema di MongoDB dengan
    field username, email, password (akan di-hash)

[ ] Backend — Install Package Auth: Install bcrypt untuk hash
    password dan jsonwebtoken untuk buat token JWT

[ ] Backend — Buat Route Auth: Tambah POST /api/auth/register
    dan POST /api/auth/login di backend

[ ] Backend — Buat Middleware Protect: Fungsi yang cek JWT token
    di setiap request yang butuh autentikasi

[ ] Frontend — Buat Halaman Login/Register: Form sederhana dengan
    email dan password, simpan JWT token ke localStorage setelah
    login berhasil

[ ] Frontend — Update Axios Instance: Tambah JWT token di header
    Authorization setiap request — gantikan sessionId dengan userId
    yang lebih permanen
